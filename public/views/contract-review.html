Contract Review

Review indexed contracts directly from a workspace.
How it works

Contract Review reads indexed matter contracts, compares them against the selected playbook and instructions, then prepares a lawyer-reviewable report with source-backed risks and negotiation points.

Upload or index
  • main agreement, schedules, statements of work, order forms, amendments, and addenda
  • policies, data processing terms, security exhibits, service levels, and pricing documents
  • counterparty drafts, prior versions, redlines, and negotiation instructions
  • client playbooks, fallback clauses, prohibited positions, and escalation rules
Output
  • key contract facts, clause extraction, and playbook comparison
  • risk matrix with severity, evidence, source anchors, and review flags
  • missing clauses, conflicts, fallback language, and escalation points
  • negotiation memo, client summary, QC notes, and agent audit trace
Standard balances coverage and speed. Detailed expands the analysis. Red-flag only focuses on material risks.
The playbook defines the clauses, standards, and risk checks used to review the contract.
Select the contracts to review. If none are selected, the run may use the available indexed contracts for this matter.
