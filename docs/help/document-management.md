# Document Management

Document management covers Add Document, View Documents, URL ingestion, folder connection, browser folder sync, workspace and matter scope, document search, and deletion.

## Add Document

Open **Add Document** from the sidebar to add files to the knowledge base.

Supported upload and indexing types include:

- PDF
- DOCX
- TXT
- CSV
- XLSX
- Markdown
- JSON
- HTML and HTM

When signed in, choose the workspace and matter before uploading. Uploading to the right scope improves document search, Contract Review, Draft, and Prep workflows.

## Uploading Files

To upload:

1. Open Add Document.
2. Select workspace and matter if available.
3. Drag files into the upload zone or click the upload area to choose files.
4. Wait for upload and indexing.

If documents do not appear in search, check that indexing completed and that the selected workspace and matter are correct.

## URL Ingestion

The Add Document screen can ingest a webpage URL as a document. The app fetches page text, stores it as a Markdown document, and indexes it.

Use URL ingestion for:

- Public guidance pages
- Public policy documents
- Online articles
- Reference pages

Do not ingest confidential third-party pages unless permitted.

## Connected Folders

The Add Document screen can connect or sync folders. Supported folder files use the same document extensions as normal uploads. Hidden files and unsupported extensions are skipped.

Folder sync is useful for:

- Matter folders on the local machine.
- Collections of contracts or pleadings.
- Evidence bundles organized by subfolder.
- Re-syncing updated files into the same matter.

Check folder permissions and the selected matter before syncing.

## View Documents

Open **View Documents** from the sidebar to browse the document library.

The screen shows:

- Document count
- Total size
- File type count
- Workspace and matter filters
- Search
- Document cards or rows
- Status information
- Delete actions

Use View Documents to confirm whether a file was uploaded, indexed, failed, or removed.

## Search and Filters

The document list includes search. Search helps locate documents by filename and metadata.

If a document is present but not used in answers, check the active Chat, Voice, Contract Review, Draft, or Prep scope, not just the document list.

## Delete Documents

Deleting a document removes it from the app's document list and retrieval context. Be careful when deleting documents used by active matters, completed runs, or prior work product.

## Best Practices

- Upload or sync documents before document-grounded chats or workflow runs.
- Use clear filenames.
- Keep documents in the correct workspace and matter.
- Split unrelated bundles where possible.
- Confirm indexing status before relying on retrieval.
- For scanned PDFs, use OCR before uploading if the app cannot extract text.
