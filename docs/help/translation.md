# Translation

The Translate feature translates pasted text or a single uploaded document and returns HTML output with review warnings, translator notes, preserved terms, and quality checks.

## Where to Find Translate

Open **Workflows** and choose **Translate**.

## Source Options

Translate supports:

- Pasted text
- One uploaded document

Supported upload types include PDF, TXT, CSV, Markdown, JSON, HTML, and HTM. For DOCX or XLSX content, add the file to Documents first or extract text if the Translate upload path does not accept that file type.

## Translate Documents

Use Upload document when the source is a single file rather than pasted text. The file translation path extracts text from the uploaded file, applies the selected source language, target language, mode, and context, then returns HTML translation output plus review notes.

## Language Selection

Choose:

- Source language, or Auto-detect.
- Target language.

The target language is required.

## Translation Modes

Modes guide translation style:

- Legal
- Business
- Technical
- Medical
- Marketing
- Academic
- Plain-language
- Literal

Use Legal mode for contracts, legal correspondence, pleadings, policies, and legal research excerpts. Use Literal mode when exact phrasing matters more than readability.

## Context

The optional context field can include:

- Jurisdiction
- Audience
- Industry
- Terminology preferences
- Document purpose
- Names or terms that should not be translated

Good context improves terminology and tone.

## Running a Translation

To translate text:

1. Open Translate from Workflows.
2. Choose Paste text.
3. Select source and target languages.
4. Choose mode.
5. Add context if useful.
6. Paste the source text.
7. Click Translate.

To translate a file:

1. Open Translate from Workflows.
2. Choose Upload document.
3. Select one supported file.
4. Select source and target languages.
5. Choose mode and context.
6. Click Translate.

## Results

Translate returns:

- HTML Translation
- Review warnings
- Translator notes
- Preserved terms
- Quality check data

Users can copy or download the HTML translation.

## Limits

Translation output should be reviewed by a qualified human for legal, medical, technical, or high-stakes use. Check names, numbers, defined terms, citations, dates, legal effect, formatting, and jurisdiction-specific terminology.
